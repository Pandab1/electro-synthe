# Rapport

Comme vous allez très vite le constater, le projet n'est pas fini. Nous nous en excusons. Cela est dû à un manque de temps et d'organisation de notre part, ainsi qu'à une trop grande ambition.

Toutes les interfaces ont été faites avec Raylib.

Pour ce qui fonctionne, le piano fonctionne correctement. C'est le coeur de notre projet. Il suffit d'aller dans l'onglet Piano et d'appuyer sur les touches A Z E R T Y U de votre clavier pour 
entendre la note correspondante et voir la courbe sur le visualiseur de fréquences.

Vous pouvez aussi changer le volume dans l'onglet Oscillator. Vous pouvez aussi y écouter la fréquence de votre choix et aussi changer la forme de l'onde.

Il est également possible de modifier l'enveloppe de la note pour le piano uniquement. Cela se fait dans l'onglet Envelope. On peut y changer le temps d'attaque, de déclin, le niveau de maintien et
le temps de relâchement. 

Il est uniquement possible d'écouter les sons avec la sortie audio de votre ordinateur, nous utilisons la librairie Miniaudio pour cela. L'écriture dans un fichier wav (au moins la partie technique) 
mais nous n'avions rien prévu pour enregistrer un son.

Concernant les commentaires dans le code, il y en a quelques uns, mais aucune vraie documentation n'a été faite faute de temps. Quant au code en lui-même il est correctement formaté grâce à un fichier 
`.clang-format`.

Nous avons sous-estimé le temps nécessaire au projet et nous avons été trop ambitieux sur ce que nous voulions faire. Nous sommes désolés.

Malgré ces problèmes, nous avons appris beaucoup de notions sur le **C** et le traitement des sons numériques en général, et nous nous sommes aussi bien amusés à faire ce projet. Nous repartons 
quand même avec de bons souvenirs.
